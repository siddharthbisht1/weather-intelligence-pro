# 🌍 Weather Intelligence Pro

An AI-Powered Environmental Intelligence Platform built with FastAPI, Machine Learning, and WebSockets.

## 🚀 Features
* **Real-time Weather & AQI:** Integration with OpenWeather API.
* **AI Predictive Modeling:** Random Forest ML models for future AQI prediction.
* **Authentication:** Secure JWT-based user login and role management.
* **Live Dashboards:** WebSocket support for real-time data streaming.
* **Automated Reporting:** Scheduled tasks and Excel/PDF report generation.

## 🛠️ Tech Stack
* **Backend:** Python, FastAPI, SQLAlchemy, SQLite
* **Machine Learning:** Scikit-Learn, Pandas
* **DevOps:** Docker, Uvicorn
* **Security:** Passlib, Bcrypt, JWT
